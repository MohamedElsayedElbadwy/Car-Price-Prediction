Car Price Prediction
This project uses machine learning to predict the market price of vehicles based on their specifications. 
It provides a data-driven approach for estimating car values accurately.
Project Overview
The goal is to analyze key features that influence car pricing and build a reliable regression model. 
The project includes everything from data exploration to a functional prediction app.

File Structure
 * CarPrice_As.csv: The primary dataset containing car details and pricing history.
 * Car_Price_Pr.ipynb: Jupyter notebook with the full data analysis, visualization, and training code.
 * model_c.pkl: The trained model file, ready for making predictions.
 * App_Car_Price.py: A Python script to run the interactive user interface.
 * Requirements.txt: List of all necessary libraries and dependencies.

How to Run
 * Clone the repo:
   git clone https://github.com/MohamedElsayedElbadwy/car_price_prediction.git

 * Install dependencies:
   pip install -r Requirements.txt

 * Launch the application:
   streamlit run App_Car_Price.py

Key Features
 * Detailed Exploratory Data Analysis (EDA).
 * Accurate price estimation using regression techniques.
 * User-friendly interface for real-time predictions.
